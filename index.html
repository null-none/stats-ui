Stats UI

Component library for football statistics visualization. Plain HTML/CSS + D3.js. All styles compiled from SCSS.

Dual comparison bars for two teams. Accepts CSV: Label,TeamA,TeamB.

Pie.render(selector, data, options)

Pass network

Team pass network on the pitch. Nodes are players, edges are passes scaled by volume.

passNetwork.draw(selector, players, passes)

Tactical flow

Line chart of tactical activity — attacking and defensive actions over time.

TacticalFlow.render(selector, data, options)

Radar

Radar (spider) chart for comparing player or team attributes. Supports two datasets.

Radar.render(selector, datasets, options)

Shot map

Shot map on the goal frame (front view). Black filled = goal, outline = saved, dashed = missed.

Shotmap.render(selector, shots, options)

Pitch map

Action dots plotted on a full pitch outline. Supports shots, tackles, interceptions, key passes, and fouls. Coordinates are 0–100 (percent of pitch length/width).

Pitchmap.render(selector, events, options)

Arrow map

Directional arrows on a full pitch — passes, runs, shots, pressing, dribbles. Arrow thickness scales with an optional value field. Curved arrows use a quadratic bezier with configurable offset.

Arrowmap.render(selector, arrows, options)

xG chart

Cumulative expected goals (xG) over match time for both teams. Each shot event is marked as a dot on the line. Dashed halftime marker at 45'.

XgChart.render(selector, data, options)

Scatter

2D scatter plot for comparing players or teams across two metrics. Supports labelled dots, color groups, custom dot sizes, and optional reference lines.

Scatter.render(selector, data, options)
